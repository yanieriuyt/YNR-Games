# YNR-Games - A Modern PSP Emulator
=========================================

**YNR-Games** is a personalized fork of PPSSPP, a fast and portable PSP emulator for Android, Windows, Mac, Linux, and iOS.

## 🚀 Getting Started

No BIOS file required to play. YNR-Games is an "HLE" emulator that balances good compatibility and speed with sensible default settings.

### For Users
- Download builds from the [Releases page](https://github.com/yanieriuyt/YNR-Games/releases)
- Check [PPSSPP's official documentation](https://www.ppsspp.org/docs) for usage guides
- View [game compatibility reports](https://report.ppsspp.org/games)

### For Developers
- See [PPSSPP Development Guide](https://www.ppsspp.org/docs/development/getting-started)
- Check the original [PPSSPP Wiki](https://github.com/hrydgard/ppsspp/wiki)
- See [DEVELOPMENT.md](DEVELOPMENT.md) for YNR-Games specific information

## 🎮 Platform Support

- **Windows** - Full support (D3D11, Vulkan, OpenGL)
- **macOS** - Full support (Metal via MoltenVK, OpenGL)
- **Linux** - Full support (Vulkan, OpenGL, SDL)
- **Android** - Full support (OpenGL ES, Vulkan)
- **iOS** - Full support (Metal)
